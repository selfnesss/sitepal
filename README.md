# SitePal

SitePal is an interactive color-palette assistant for website concepts.

The first version includes:

- prompt-based local palette generation;
- clickable palettes that recolor the whole interface;
- live website preview with desktop, tablet, and mobile modes;
- visual demonstrations for golden ratio, rule of thirds, contrast, and banner blindness;
- a compact checklist based on core web design rules.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The main app lives in `src/main.tsx`, with styles in `src/styles.css`.
